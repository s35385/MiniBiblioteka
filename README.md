# Mini Biblioteka – projekt Java


## Zawarte klasy

`Ksiazka`

Reprezentuje książkę:

- tytuł, autor, liczba stron

- informacja czy dostępna

- metody: `wypiszInfo()`, `wypozycz()`, `zwroc()`

`Czytelnik`

Reprezentuje użytkownika biblioteki:

- imię, nazwisko, numer karty

- liczba wypożyczonych książek

- metody: `wypiszDane()`, `zwiekszLiczbeWypozyczen()`, `zmniejszLiczbeWypozyczen()`

`Biblioteka`

Przechowuje tablicę książek:

- dodawanie książek

- wyszukiwanie po tytule

- wypisanie dostępnych

- policzenie dostępnych

- wypożyczenie i zwrot książki


`Main`

Zawiera scenariusz testowy:

1. utworzenie kilku książek i czytelników  

2. dodanie książek do biblioteki  

3. wypisanie dostępnych książek  

4. wypożyczenie i zwrot  

5. ponowne wypisanie stanu biblioteki
